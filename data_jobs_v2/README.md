## Dashboard Overview

This second version of the Data Jobs Dashboard condenses the information for the user down into one page designed to give job seekers the most critical insights. 

Users are also able to select their job title along with their job location using the slicers at the top of the report. This example shows information for Data Analysts jobs in the United States. 

![Alt view of project 2](/images/project2_page2.png)

This single report page acts as a hub for users in the data job market. It showcases KPIs like Job Count, Skills per Job, Median Yearly/Hourly Salary. You can also see the Skill Popularity and compare Salaries accross different Job Titles.

## Conclusion
In this project, I've made a few decisions for the user in order to simplify the data shows to give more attention to the information the user would care about. Users are able to slice/filter for specific information they'd like to see without having to know how to navigate the report. 

Overall, the more advanced side of Power BI makes things much more efficient while also providing a cleaner UX/UI for potential job seekers.
